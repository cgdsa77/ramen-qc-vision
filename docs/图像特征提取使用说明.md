# 图像特征提取使用说明

## 📋 概述

图像特征提取模块 (`src/features/image_feature_extractor.py`) 已实现，可以从检测框区域提取真实的图像特征，用于更准确的评分。

## ✨ 已实现的功能

### 1. `noodle_rope`（面条）特征提取

- **厚度（thickness）**：
  - 基于检测框的宽高比和边缘密度
  - 计算面条的平均宽度估算值
  
- **光泽度（gloss）**：
  - 转换为HSV色彩空间，分析V通道（亮度）
  - 高光泽 = 高平均亮度 + 低亮度方差（均匀明亮）
  
- **完整性（integrity）**：
  - 使用轮廓分析检测是否有断裂
  - 完整的面条应该有连续的大轮廓
  
- **弹性（elasticity）**：
  - 当前使用完整性和光泽度的组合作为代理
  - 未来可基于序列帧分析形变程度

### 2. `hand`（手部）特征提取

- **位置（position）**：
  - 检测框中心相对于图像的位置
  - 理想位置：画面中心偏下（操作区域）
  
- **角度（angle）**：
  - 使用轮廓的最小外接矩形计算主轴方向
  - 或基于检测框的宽高比估算角度
  
- **动作（action）**：
  - 当前使用位置和角度的组合作为代理
  - 未来可基于序列帧分析运动轨迹
  
- **协调性（coordination）**：
  - 当前使用位置、角度、动作的平均值作为代理
  - 未来可分析多手检测的对称性

### 3. `noodle_bundle`（面束）特征提取

- **紧实度（tightness）**：
  - 计算边缘密度和轮廓面积占比
  - 紧实 = 高边缘密度 + 高轮廓面积占比
  
- **均匀度（uniformity）**：
  - 将ROI分成3x3网格
  - 计算各网格区域的边缘密度方差
  - 方差越小，越均匀

## 🔧 使用方法

### 在评分器中使用

评分器 (`StretchScorer`) 已自动集成图像特征提取功能：

```python
from src.scoring.stretch_scorer import StretchScorer

# 初始化评分器（默认使用图像特征）
scorer = StretchScorer(use_image_features=True)

# 评分视频（传入视频路径以提取特征）
result = scorer.score_video(video_detections, video_path="path/to/video.mp4")
```

### 直接使用特征提取器

```python
from src.features.image_feature_extractor import ImageFeatureExtractor
import cv2

# 初始化提取器
extractor = ImageFeatureExtractor()

# 读取图像
image = cv2.imread("path/to/image.jpg")

# 定义检测结果
detection = {
    'class': 'noodle_rope',
    'xyxy': [100, 100, 300, 400],  # [x1, y1, x2, y2]
    'conf': 0.9
}

# 提取特征
features = extractor.extract_features(image, detection)
print(features)
# 输出: {'thickness': 3.2, 'gloss': 4.1, 'integrity': 3.8, 'elasticity': 3.95}
```

## ⚠️ 注意事项

1. **特征值范围**：
   - 提取的原始特征值范围约为 0-5
   - 评分器会使用阈值将其映射到1-5分的整数评分

2. **回退机制**：
   - 如果图像特征提取失败，评分器会自动回退到基于置信度的评分
   - 如果未提供视频路径或图像，也会使用置信度评分

3. **性能考虑**：
   - 图像特征提取会增加计算时间
   - 对于实时应用，可以考虑降低处理频率或使用GPU加速

4. **特征准确性**：
   - 当前实现是基础版本，特征提取算法可以进一步优化
   - 建议通过实际测试验证特征值的合理性
