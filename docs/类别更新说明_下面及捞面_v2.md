## 🎯 新的类别定义

### 类别顺序（重要！）

```
0: noodle_rope   # 面条（正在捞出、已捞出等情况）
1: hand          # 手
2: tools_noodle  # 工具（漏勺等）
3: soup_noodle   # 汤中面条（在汤锅中的面条）
```

### 类别详细说明

1. **noodle_rope（面条）**
   - 标注正在捞出、已捞出在漏勺中、或其他情况下的面条
   - **重要区分**：在汤锅中的面条应标注为`soup_noodle`，不要标注为`noodle_rope`

2. **hand（手）**
   - 标注操作者的手部
   - 包括所有操作中的手部（搅拌、检查、捞面等）
   - 标注整个手部区域（包括手腕）

3. **tools_noodle（工具）**
   - 标注用于捞面的工具（漏勺、筷子等）
   - 标注整个工具的可见部分

4. **soup_noodle（汤中面条）** ← 新增
   - 标注在汤锅中的面条
   - 用于区分汤中的面条和其他情况下的面条
   - 帮助模型更好地理解不同场景下的面条状态

---

## ⚠️ 重要标注规则

### 面条标注的区分

**标注为 soup_noodle（汤中面条）**：
- 面条在汤锅中
- 面条完全或大部分浸泡在汤中
- 面条正在锅中煮制

**标注为 noodle_rope（面条）**：
- 面条正在被捞出
- 面条已经在漏勺中
- 面条已经从锅中捞出
- 其他不在汤中的面条

### 标注要点

1. **严格区分**：根据面条的位置和状态选择正确的类别
2. **类别顺序**：标注时必须严格按照类别顺序（0, 1, 2, 3）
3. **一致性**：确保所有视频的标注标准一致
