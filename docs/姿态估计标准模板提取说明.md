# 姿态估计标准模板提取说明

## 概述

本功能用于从标准视频（cm1-cm3）中自动提取姿态关键点序列，建立标准姿态模板。**无需手动标注**，使用预训练模型（MediaPipe）自动提取。

## 为什么不需要手动标注？

1. **预训练模型效果好**：MediaPipe/MMPose等预训练模型在大规模数据上训练，对于人体姿态估计效果很好
2. **标注成本高**：手动标注姿态关键点（17+个关键点）比标注检测框复杂得多，耗时且容易出错
3. **自动化高效**：自动提取可以在几分钟内完成，而手动标注可能需要数小时
4. **可以先验证**：先用预训练模型提取，验证效果后再决定是否需要自定义标注

## 功能说明

### 主要功能

1. **自动提取关键点序列**：从cm1-cm3视频中提取每一帧的人体姿态关键点
2. **计算统计信息**：计算关键点位置、角度的均值、标准差等统计信息
3. **建立标准模板**：将统计信息保存为标准模板，用于后续的动作规范性评分

### 提取的关键点

使用MediaPipe姿态估计模型，提取33个关键点，包括：

- **头部**：鼻子、眼睛、耳朵等
- **手臂**：肩膀、肘部、手腕、手指等（最重要的部分）
- **躯干**：肩膀、臀部
- **腿部**：膝盖、脚踝等

### 计算的关键角度

- **左臂角度**：左肩-左肘-左腕的角度
- **右臂角度**：右肩-右肘-右腕的角度
- **左肩角度**：左肘-左肩-右肩的角度
- **右肩角度**：右肘-右肩-左肩的角度

## 使用方法

### 1. 安装依赖

确保已安装MediaPipe：

```bash
pip install mediapipe
```

### 2. 运行提取脚本

```bash
python scripts/extract_pose_template_cm1_cm3.py
```

### 3. 输出文件

脚本会生成两个文件：

1. **标准模板（统计信息）**：
   - 文件路径：`data/scores/抻面/pose_template_cm1_cm3.json`
   - 包含内容：
     - 关键点统计信息（均值、标准差、范围等）
     - 角度统计信息（均值、标准差、范围等）
     - 整体检测率统计

2. **完整序列（详细数据）**：
   - 文件路径：`data/scores/抻面/pose_sequences_cm1_cm3.json`
   - 包含内容：
     - 每一帧的关键点位置（仅保存手臂相关关键点以节省空间）
     - 时间戳信息
     - 检测状态

## 效果验证

### 检查姿态检测率

运行脚本后，会显示：
- 总采样帧数
- 检测到姿态的帧数
- 姿态检测率（建议>80%）

如果检测率太低，可能的原因：
1. 视频中人物太小或被遮挡
2. 光线条件不好
3. 需要调整姿态估计模型参数

### 查看关键点统计

打开 `pose_template_cm1_cm3.json`，检查：

1. **关键点位置是否合理**：
   - 肩膀、肘部、手腕的位置是否符合预期
   - x、y坐标范围是否在图像范围内

2. **置信度是否足够**：
   - 检查 `confidence.mean` 是否>0.5（建议>0.7）
   - 如果置信度过低，说明检测不够稳定

3. **角度统计是否合理**：
   - 左臂角度、右臂角度的范围是否符合抻面动作的特点
   - 标准差的数值是否合理（太小可能动作单调，太大可能动作不规范）

## 参数调整

如果需要调整提取参数，可以修改脚本中的：

- **sample_fps**：采样帧率（默认2.0，即每秒提取2帧）
  - 增加采样率可以获得更多数据，但处理时间更长
  - 减少采样率可以加快处理，但可能丢失细节

- **min_detection_confidence**：最小检测置信度（在`src/models/pose.py`中调整）
  - 默认0.5
  - 提高可以过滤低质量检测，但可能减少样本数

## 后续步骤

提取完成后，建议：

1. **验证数据质量**：
   - 检查检测率、置信度、关键点位置是否合理
   - 如有问题，调整参数后重新提取

2. **可视化验证**（可选）：
   - 可以编写脚本在视频帧上绘制关键点，人工检查效果
   - 确认关键点位置准确

3. **逐步推进**：
   - 确认标准模板质量后，再进行后续的动作规范性评分开发
   - 根据实际效果不断调整和优化

## 常见问题

### Q1: 检测率太低怎么办？

**A**: 尝试以下方法：
1. 降低 `min_detection_confidence` 阈值
2. 确保视频中人物清晰可见
3. 检查视频分辨率是否足够

### Q2: 关键点位置不准确怎么办？

**A**: 这是预训练模型的局限性，可以：
1. 先用现有结果进行后续开发
2. 如果效果确实不好，再考虑使用更精确的模型（如MMPose）
3. 或者对关键帧进行少量手动标注，作为校准

### Q3: 需要重新提取吗？

**A**: 可以随时重新运行脚本，会覆盖之前的结果。建议在以下情况重新提取：
1. 调整了参数
2. 添加了新的标准视频
3. 发现数据质量问题

## 技术细节

### 使用的模型

- **MediaPipe Pose**：Google开发的人体姿态估计模型
- **模型复杂度**：Medium（model_complexity=1）
- **检测置信度阈值**：0.5
- **跟踪置信度阈值**：0.5

### 关键点格式

每个关键点包含：
- `x`: x坐标（像素）
- `y`: y坐标（像素）
- `z`: 深度信息（相对值）
- `confidence`: 置信度（0-1）
- `name`: 关键点名称

### 角度计算

使用三点角度计算法：
- 输入：三个关键点坐标
- 输出：角度（度数，0-180度）
