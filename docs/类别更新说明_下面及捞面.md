## 🎯 新的类别定义

### 类别顺序（重要！）

```
0: noodle_rope   # 面条
1: hand          # 手
2: tools_noodle  # 工具
```

### 类别说明

1. **noodle_rope（面条）**
   - 标注所有可见的面条区域
   - 包括锅中、正在捞出、或已捞出在漏勺中的面条

2. **hand（手）**
   - 标注操作者的手部
   - 包括所有操作中的手部（搅拌、检查、捞面等）
   - 标注整个手部区域（包括手腕）

3. **tools_noodle（工具）**
   - 标注用于捞面的工具（漏勺、筷子等）
   - 标注整个工具的可见部分

## ⚠️ 重要提醒

1. **重新标注**：由于类别已更改，所有视频都需要使用新的3类别定义重新标注

2. **类别顺序**：标注时必须严格按照新的类别顺序：
   - 0 = noodle_rope
   - 1 = hand
   - 2 = tools_noodle

3. **不再标注pot**：不要再标注锅（pot），即使画面中有锅也不需要标注

4. **一致性检查**：标注前和标注过程中，确保所有视频子目录的classes.txt与主文件一致
