## 问题1：是否需要标注姿态（骨骼关键点）？

### 答案：**不一定需要手动标注**

**原因：**
1. **可以使用预训练模型**：现有的人体姿态估计模型（如RTMPose、MMPose、MediaPipe）已经训练好，可以直接使用
2. **手动标注成本高**：标注姿态关键点（17个或更多关键点）比标注目标检测框更复杂、耗时
3. **预训练模型效果更好**：大模型在大量数据上训练，通常比小数据集训练的模型效果更好

### 推荐方案：

**方案A：使用预训练姿态估计模型（推荐）**
- 使用 RTMPose 或 MMPose 的预训练模型
- 直接对视频进行姿态估计，无需手动标注
- 基于估计结果分析动作规范性

**方案B：如果有自定义需求，可以标注姿态**
- 使用 LabelMe 等工具标注关键点
- 训练自定义姿态估计模型
- 适用于有特殊关键点需求的情况

**建议：先使用预训练模型，如果有特殊需求再考虑标注**
