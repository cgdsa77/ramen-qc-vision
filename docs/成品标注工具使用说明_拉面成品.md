# 拉面成品标注工具使用说明

## 用途

对 **兰州拉面成品** 图片进行人工标注，形成标准数据集，用于成品分析评估（面条质感、汤型、辅料等）。标注结果保存在 `data/scores/拉面成品/annotations.json`。

## 打开方式

1. 启动本机 Web 服务（在项目根目录执行）：
   ```bash
   python start_web.py
   ```
2. 浏览器访问：
   ```
   http://127.0.0.1:8000/product-annotation
   ```
   或从首页进入「兰州拉面成品评估」入口（若已添加入口）。

## 操作步骤

1. **左侧列表**：显示 `data/raw/拉面成品/` 下所有 `lmcp.*.jpg` 图片，已标注的会显示 ✓。
2. **点击某张图**：中间显示大图，右侧显示表单；若该图已有标注，会自动填入。
3. **必填项**：
   - **面条质感**：优秀 / 良好 / 一般 / 较差 / 差
   - **汤型**：红汤（有辣椒）/ 清汤（无辣椒）
   - **面条裹辣椒情况**：所有 / 部分 / 无
4. **选填项**：牛肉、鸡蛋、蒜苗碎（无/少量/适量/较多/无法判断）、备注。
5. **保存**：点击「保存当前标注」会将该张图的标注写入汇总，并更新 `annotations.json`（包含所有已标注图片的记录）。

## 数据文件

- **图片目录**：`data/raw/拉面成品/`（如 `lmcp.1.jpg`）
- **标注汇总**：`data/scores/拉面成品/annotations.json`  
  - 结构：`{ "items": [ { "image": "lmcp.1.jpg", "noodle_quality": "good", ... }, ... ], "updated": "..." }`
