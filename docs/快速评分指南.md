# 快速评分指南

## 问题：手动评分太繁琐

手动逐帧评分确实工作量很大。本指南提供几种高效评分方法，可以将工作量减少80-90%。

## 方法1：抽样评分（推荐，最简单）

**思路**：不需要对每一帧都评分，只对关键帧进行评分

### 步骤：

1. **选择关键帧**：
   ```bash
   python scripts/batch_scoring_helper.py --action select --video cm1 --interval 10
   ```
   - `--interval 10`：每10帧选1帧
   - 结果会保存到 `data/scores/抻面/cm1/key_frames.json`

2. **只对关键帧进行评分**：
   - 打开评分工具：http://localhost:8000/scoring-tool
   - 只评分关键帧列表中的帧
   - 工作量减少90%（从90帧减少到9帧）

3. **建立评分标准**：
   - 基于关键帧的评分建立标准数据集
   - 其他帧通过插值或规则推断

### 优势：
- ✅ 工作量减少80-90%
- ✅ 仍能建立有效的评分标准
- ✅ 快速完成标准数据集

## 方法2：模板评分（最快速）

**思路**：对典型帧详细评分，然后自动应用到相似帧

### 步骤：

1. **详细评分1-2个典型帧**：
   - 选择有代表性的帧（如cm1_00010.jpg）
   - 进行详细评分（所有属性都评分）

2. **创建评分模板**：
   ```bash
   python scripts/batch_scoring_helper.py --action template --video cm1 --template-frame cm1_00010.jpg --threshold 0.8
   ```
   - 会自动找到相似帧（相似度>=0.8）

3. **应用模板评分**：
   ```bash
   python scripts/batch_scoring_helper.py --action apply --video cm1 --template-frame cm1_00010.jpg
   ```
   - 自动将模板评分应用到所有相似帧

### 优势：
- ✅ 工作量减少95%以上
- ✅ 只需评分1-2个典型帧
- ✅ 自动批量处理

## 方法3：简化评分维度（最实用）

**思路**：减少评分维度，只关注核心指标

### 实施：

在评分工具中，可以只评分：
- **面条（noodle_rope）**：只评分"粗细"和"完整程度"（2个维度）
- **手部（hand）**：只评分"位置"和"动作"（2个维度）

其他维度（弹性、光泽、角度、协调性）可以通过规则或统计特征自动计算。

### 优势：
- ✅ 每个检测框的评分时间减少50%
- ✅ 保持评分有效性
- ✅ 快速完成评分

## 推荐工作流程

### 快速建立评分标准（1-2小时）：

1. **第一步**：抽样评分
   ```bash
   # 对cm1, cm2, cm3各选择10个关键帧
   python scripts/batch_scoring_helper.py --action select --video cm1 --interval 10
   python scripts/batch_scoring_helper.py --action select --video cm2 --interval 10
   python scripts/batch_scoring_helper.py --action select --video cm3 --interval 10
   ```

2. **第二步**：只评分关键帧
   - 打开评分工具
   - 只评分关键帧（约30帧，而不是300帧）
   - 使用简化维度（只评分核心属性）

3. **第三步**：建立模板并批量应用
   ```bash
   # 为每个视频创建1-2个模板
   python scripts/batch_scoring_helper.py --action template --video cm1 --template-frame cm1_00010.jpg
   python scripts/batch_scoring_helper.py --action apply --video cm1 --template-frame cm1_00010.jpg
   ```

### 预期效果：

- **原始工作量**：300帧 × 4个检测框 × 4个属性 = 4800次评分
- **优化后工作量**：30个关键帧 × 4个检测框 × 2个核心属性 = 240次评分
- **工作量减少**：95%

## 工具使用示例

### 示例1：选择关键帧
```bash
# 每10帧选1帧
python scripts/batch_scoring_helper.py --action select --video cm1 --interval 10

# 每5帧选1帧（更密集）
python scripts/batch_scoring_helper.py --action select --video cm1 --interval 5
```

### 示例2：创建模板
```bash
# 基于已评分的帧创建模板
python scripts/batch_scoring_helper.py --action template --video cm1 --template-frame cm1_00010.jpg --threshold 0.8
```

### 示例3：应用模板
```bash
# 将模板评分应用到相似帧
python scripts/batch_scoring_helper.py --action apply --video cm1 --template-frame cm1_00010.jpg
```

## 注意事项

1. **抽样间隔选择**：
   - 动作变化快的视频：interval=5（更密集）
   - 动作变化慢的视频：interval=10或15（更稀疏）

2. **模板相似度阈值**：
   - 阈值越高，匹配的帧越少，但质量越高
   - 建议阈值：0.7-0.8

3. **评分一致性**：
   - 建议多人评分后取平均值
   - 或使用模板保持一致性
