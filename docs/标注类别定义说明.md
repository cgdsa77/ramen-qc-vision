# 标注类别定义说明

## 三个类别的定义

### 1. hand（手）
- **定义**：厨师的手部，包括手掌、手指等
- **标注要求**：
  - 只标注手部可见的部分
  - 如果手被遮挡，只标注可见的部分
  - 通常每个视频帧中会有2个hand（左右手）

### 2. noodle_rope（面条绳/拉长的面条）
- **定义**：**被拉伸拉长的面条，呈现条状/绳状**，通常是细长的面条条
- **特征**：
  - 细长的条状
  - 已经被拉伸、拉长
  - 看起来像绳子或细长的面条条
  - 通常比较细，宽度相对较小
- **典型场景**：
  - 厨师双手拉开面条时，中间被拉长的部分
  - 悬在空中或接近拉伸状态的面条条
  - 细长的面条段

**示例**：在抻面过程中，双手向两边拉开，中间被拉成细长条状的面条就是 noodle_rope

### 3. noodle_bundle（面条束/面条团）
- **定义**：**成束、成团的面条**，通常是多个面条条聚集在一起，或者尚未完全拉伸的面条团
- **特征**：
  - 多个面条条聚集在一起
  - 看起来像一束或一团
  - 通常较粗，宽度相对较大
  - 尚未被完全拉伸开
  - 或者多个细面条条聚集在一起
- **典型场景**：
  - 双手握住的、尚未完全拉伸的面条束
  - 多个面条条聚集在一起的区域
  - 较粗的面条团/束

**示例**：在抻面过程中，厨师双手握住的面条部分（较粗，多个面条条聚集），或者尚未拉伸的面条团就是 noodle_bundle

## 区分要点

### noodle_rope vs noodle_bundle 的关键区别

| 特征 | noodle_rope | noodle_bundle |
|------|-------------|---------------|
| **形状** | 细长的条状/绳状 | 较粗的束状/团状 |
| **宽度** | 相对较细 | 相对较粗 |
| **状态** | 已经被拉伸开 | 聚集在一起，尚未完全拉伸 |
| **数量** | 通常是单个细长的面条条 | 通常是多个面条条聚集 |
| **位置** | 通常在双手之间的中间部分 | 通常在双手握住的端部 |

### 实际标注时的判断逻辑

1. **如果面条看起来很细长，像绳子一样被拉长** → `noodle_rope`
2. **如果面条看起来较粗，多个面条条聚集在一起，像一束或一团** → `noodle_bundle`
3. **如果面条被双手握住，看起来较粗** → 通常是 `noodle_bundle`
4. **如果面条在双手之间，被拉伸得很细长** → 通常是 `noodle_rope`

## 标注建议

### 标注顺序建议

1. **先标注hand（手）**：
   - 确保手部被正确标注
   - 每个可见的手部都要标注

2. **然后标注noodle_bundle（面条束）**：
   - 双手握住的面条端部
   - 较粗的面条团/束

3. **最后标注noodle_rope（面条绳）**：
   - 被拉伸的细长面条条
   - 双手之间的中间部分

### 常见情况处理

**情况1：双手拉开面条**
- 左右手各标注一个 `hand`
- 双手握住的部分标注 `noodle_bundle`
- 中间被拉长的部分标注 `noodle_rope`

**情况2：面条全部被拉伸开**
- 可能只有 `noodle_rope`，没有 `noodle_bundle`

**情况3：面条刚开始，尚未拉伸**
- 可能只有 `noodle_bundle`，没有 `noodle_rope`

**情况4：面条被打成结或缠绕**
- 缠绕聚集的部分标注 `noodle_bundle`
- 细长的部分标注 `noodle_rope`

## 注意事项

⚠️ **重要提示**：
- 不要将 `noodle_rope` 和 `noodle_bundle` 混淆
- 同一帧图片中，`noodle_rope` 和 `noodle_bundle` 可能同时存在
- 根据实际情况判断，不要生搬硬套
- 如果不确定，可以参考同一视频中其他帧的标注模式

## 示例说明

### 典型的抻面动作场景：

**帧1（开始阶段）**：
- 2个 `hand`（左右手）
- 1-2个 `noodle_bundle`（双手握住的面条端部，较粗）
- 可能没有 `noodle_rope`（尚未拉伸）

**帧2（拉伸中）**：
- 2个 `hand`（左右手）
- 2个 `noodle_bundle`（左右手握住的面条端部）
- 1-2个 `noodle_rope`（中间被拉伸的细长部分）

**帧3（完全拉伸）**：
- 2个 `hand`（左右手）
- 可能只有 `noodle_rope`（面条完全被拉伸开，很细长）
- 可能没有或很少 `noodle_bundle`

## 总结

- **noodle_rope** = 细长的面条条（被拉伸开的）
- **noodle_bundle** = 较粗的面条束/团（聚集在一起的）

记住：**细长=rope，粗束=bundle**
