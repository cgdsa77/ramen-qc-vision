## 使用步骤

### 步骤1: 启动Web服务
确保Web服务正在运行：
```bash
python start_web.py
```

### 步骤2: 打开评分工具
方式1：双击运行 `打开评分工具.bat`
方式2：浏览器访问 `http://localhost:8000/scoring-tool`

### 步骤3: 选择视频并评分
1. 在"选择视频"下拉框中选择要评分的视频（cm4, cm5, cm6, cm7）
2. **确保"仅显示关键帧"复选框已勾选**（默认已勾选）
3. 系统会自动加载该视频的10个关键帧
4. 使用"上一帧"/"下一帧"按钮浏览关键帧
5. 为每个关键帧的检测项（noodle_rope, hand, noodle_bundle）进行评分：
   - **noodle_rope**: 粗细、弹性、光泽、完整性（1-5分）
   - **hand**: 位置、动作、角度、协调性（1-5分）
   - **noodle_bundle**: 紧实度、均匀度（1-5分）
6. 填写备注（可选）
7. 点击"保存评分"按钮

### 步骤4: 查看评分进度
运行以下命令查看评分进度：
```bash
python scripts/manual_scoring_helper.py --action progress --stage 抻面
```

## 评分标准

参考文档：`docs/评分标准定义指南.md`

### noodle_rope（面条）评分标准：
- **粗细**: 1-5分，评估面条的粗细程度
- **弹性**: 1-5分，评估面条的弹性表现
- **光泽**: 1-5分，评估面条的光泽度
- **完整性**: 1-5分，评估面条是否完整无断裂

### hand（手部）评分标准：
- **位置**: 1-5分，评估手部位置是否合适
- **动作**: 1-5分，评估动作是否标准
- **角度**: 1-5分，评估手部角度是否合适
- **协调性**: 1-5分，评估双手协调性

### noodle_bundle（面条束）评分标准：
- **紧实度**: 1-5分，评估面条束的紧实程度
- **均匀度**: 1-5分，评估面条束的均匀程度

## 评分完成后

### 1. 重新生成评分规则
评分完成后，运行以下命令重新生成评分规则和阈值：
```bash
python scripts/build_scoring_rules.py
```

### 2. 验证评分效果
重新启动Web服务，上传视频测试自动评分效果。

## 注意事项

1. **关键帧选择策略**：
   - 均匀分布在整个视频中
   - 优先选择有检测框的帧（特别是hand和noodle_rope）
   - 每个视频选择10个关键帧

2. **评分一致性**：
   - 尽量保持评分标准一致
   - 可以参考cm1~cm3的评分作为参考

3. **数据保存**：
   - 评分数据自动保存到 `data/scores/抻面/{视频名}/{帧名}_scores.json`
   - 建议定期备份评分数据

4. **如果cm8~cm12有标注数据**：
   - 运行 `python scripts/select_keyframes_for_scoring.py` 重新选择关键帧
   - 然后按照上述步骤进行评分
