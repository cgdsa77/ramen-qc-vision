## 二、登录页专属动画效果

适配登录页核心元素：背景图片墙、登录卡片、输入框、登录按钮、标题文字。

1. **背景图片墙呼吸微动效**

    - 适用场景：登录页背景的兰州拉面图片矩阵

    - 效果说明：背景图片做缓慢、随机的微缩放+轻微位移动画，每张图片错开动画启动时间，形成低干扰的呼吸感动态背景，不抢登录卡片的视觉焦点

    - 核心API：`keyframes`、`stagger`、`loop`

    - 实现提示：loop循环keyframes动画，每张图片scale在11.03之间循环，y轴位移03px，动画时长8-12秒随机，透明度保持0.6-0.8，不干扰登录核心操作

    - 价值：弱化静态背景的单调感，贴合兰州拉面产品主题，营造沉浸式登录氛围

2. **登录卡片入场弹动动画**

    - 适用场景：登录页居中的登录卡片

    - 效果说明：页面加载时，登录卡片从屏幕正上方-50px位置掉落，配合弹簧动画轻微回弹后稳定在居中位置，同步透明度0→1过渡，比普通渐入更有记忆点

    - 核心API：`spring`

    - 实现提示：spring设置刚度80、阻尼12，初始y值-50，最终y值0，与透明度动画同步执行，页面加载完成后立即触发

    - 价值：登录页加载第一时间抓住用户注意力，符合物理直觉，提升界面精致感，不浮夸且有辨识度

3. **输入框聚焦交互+错误反馈动画**

    - 适用场景：账号、密码输入框

    - 效果说明：聚焦时，输入框边框高亮过渡，整体轻微向上位移2px，输入文字字间距轻微拉开；失焦时平滑回弹；输入错误时，触发左右抖动动画做错误提示

    - 核心API：`animate`、`spring`、`keyframes`

    - 实现提示：聚焦动画时长200ms，ease用easeOut；错误反馈用keyframes左右位移（-5px → 5px → -3px → 3px → 0），时长300ms

    - 价值：输入操作有明确视觉反馈，错误提示直观，降低输入失误率

4. **登录按钮加载状态动画**

    - 适用场景：登录按钮点击后的请求加载状态

    - 效果说明：点击登录后，按钮宽度平滑收缩为圆形加载态，内部文字淡出，用数值动画做环形加载进度；登录成功/失败后，按钮回弹到原始宽度，同步触发状态反馈动画

    - 核心API：`animate`、`spring`、`keyframes`

    - 实现提示：点击后先animate收缩按钮宽度，同步隐藏文字，再用数值动画驱动stroke-dashoffset实现环形加载，结果返回后触发回弹动画

    - 价值：登录请求过程有明确状态反馈，避免用户重复点击，提升系统专业感与操作确定性

## 三、首页（功能主页面）专属动画效果

适配首页核心元素：顶部用户栏、系统大标题、模块标题、功能卡片、卡片内进入按钮。

1. **主标题打字渐显入场动画**

    - 适用场景：首页顶部「面向兰州拉面制作的标准化流程视觉质检系统」大标题

    - 效果说明：页面加载后，标题文字逐个打字出现，配合透明度0→1+轻微y轴位移动画；主标题动画完成后，副标题（抻面·下面·捞面全流程检测...）淡入，整体流畅不拖沓

    - 核心API：`stagger`、`animate`、`keyframes`

    - 实现提示：用stagger拆分每个文字，错开50ms执行animate动画，每个文字从opacity:0、y:10px到opacity:1、y:0，时长300ms，ease用easeOut

    - 价值：首页加载第一时间传递系统核心主题，打字动画提升仪式感，强化系统专业定位

2. **功能卡片hover悬浮联动动画**

    - 适用场景：首页所有功能卡片（抻面动作检测、下面及捞面检测、标注工具等全量卡片）

    - 效果说明：hover时，卡片整体向上悬浮（y轴-8px），阴影加深放大，卡片内标题文字颜色高亮，「进入」按钮同步触发hover放大动画，背景做轻微暖色调渐变；取消hover后平滑回弹，全程弹簧动画

    - 核心API：`spring`、`animate`

    - 实现提示：hover触发spring动画，y轴0→-8px，刚度100、阻尼14，阴影blur从8px到20px，同步联动卡片内按钮、文字的动画

    - 价值：大幅提升卡片的可交互感知，让静态功能模块更有吸引力，引导用户点击进入对应功能，同时保持界面沉稳专业的调性

3. **模块标题锚定生长动画**

    - 适用场景：「流程检测」「标注工具」「综合评分」模块标题

    - 效果说明：当模块滚动到视口顶部20%位置时，模块标题触发下划线从左到右的生长动画（宽度0→100%），同时标题文字放大1.05倍高亮；滚动离开后，下划线收缩，文字回弹

    - 核心API：`scroll`、`animate`、`viewport`

    - 实现提示：用scroll监听模块位置，进入指定视口区域时，animate驱动下划线width从0到100%，时长400ms，ease用easeOut

    - 价值：强化页面模块层级，滚动时明确告知用户当前浏览区域，提升长页面的导航感

4. **卡片「进入」按钮跟随引导动画**

    - 适用场景：所有卡片内的「进入」按钮，与卡片hover联动

    - 效果说明：卡片hover时，按钮除基础缩放外，同步触发按钮内箭头（可补充）从左到右位移3px，背景色饱和度提升；取消hover时所有属性回弹，与卡片动画完全联动

    - 核心API：`spring`、`animate`

    - 实现提示：与卡片hover事件绑定，弹簧参数与卡片保持一致，形成联动感

    - 价值：强化按钮的点击引导，让用户一眼锁定操作入口，提升界面操作指引性

## 四、子模块页面通用动画效果

适配所有功能子页面：抻面动作检测、标注工具、实时监测、评分可视化等模块，覆盖视频上传、检测结果、数据可视化、标注操作等核心场景。

1. **数据面板数值滚动动画**

    - 适用场景：子模块内的检测评分、数值指标、进度条、合格率等数据展示

    - 效果说明：检测结果加载完成后，面板内数值从0顺滑滚动到最终数值，配合弹簧动画，进度条同步从0到目标宽度，数值最终停留时有轻微弹动反馈

    - 核心API：`spring`、`animate`、`interpolate`

    - 实现提示：用animate驱动数值从0到目标值，spring设置阻尼18、刚度80，配合interpolate映射数值到进度条宽度，实时更新DOM数值

    - 价值：数据展示更有仪式感，让用户直观感知数值变化，强化质检系统的专业数据属性

2. **功能区块按流程入场动画**

    - 适用场景：子模块内的视频上传区、操作按钮区、检测结果可视化区、骨骼可视化面板

    - 效果说明：进入子模块后，按操作流程顺序入场：先上传区→再操作按钮→最后结果展示区；检测完成后，结果面板从右往左滑入+渐显，内部可视化图表同步触发绘制动画

    - 核心API：`stagger`、`spring`、`animate`

    - 实现提示：用stagger控制区块入场间隔200ms，每个区块从opacity:0、x:20px到opacity:1、x:0，spring动画过渡

    - 价值：子模块操作流程更清晰，引导用户按步骤操作，强化检测完成的成就感

3. **骨骼/动作可视化实时平滑动画**

    - 适用场景：抻面动作检测的手部骨骼可视化、动作捕捉实时渲染模块

    - 效果说明：用popmotion实时数值动画，驱动骨骼节点的位置跟随视频检测结果平滑更新，节点连线同步过渡，每个动作变化用弹簧动画柔化，避免骨骼跳动的生硬感

    - 核心API：`spring`、`animate`、`pointer`

    - 实现提示：检测到骨骼坐标变化时，用spring动画驱动节点x/y值从旧坐标过渡到新坐标，刚度60、阻尼10，保证动作丝滑

    - 价值：核心的骨骼可视化功能更流畅，大幅提升系统的技术感与专业度，直观呈现动作检测核心价值

4. **步骤/标签页切换转场动画**

    - 适用场景：标注工具的多步骤操作、子模块内多标签页切换、流程步骤切换

    - 效果说明：切换时，当前内容向左滑出渐隐，新内容从右滑入渐显，配合弹簧动画；底部标签激活下划线同步滑动到对应位置，转场无割裂感

    - 核心API：`spring`、`animate`、`keyframes`

    - 实现提示：切换时，animate控制旧内容opacity:1→0、x:0→-30px，新内容opacity:0→1、x:30→0，同步执行下划线translateX动画

    - 价值：多步骤操作更流畅，切换有明确空间感，避免页面跳转的生硬感，提升操作连贯性

## 五、微交互细节动画（全页面补充，提升精致度）

1. **复选框/单选框勾选路径动画**

    - 适用场景：标注工具的标注项选择、系统设置的选项勾选

    - 效果说明：勾选时，用路径动画绘制SVG对勾，stroke-dashoffset从100%到0，同步触发复选框缩放弹动；取消勾选时反向执行

    - 核心API：`animate`、`keyframes`

    - 价值：表单选择反馈更细腻，提升界面精致度，强化操作确定性

2. **通知提示框滑入滑出动画**

    - 适用场景：系统成功/错误/警告提示（标注完成、检测成功、操作失败等）

    - 效果说明：提示出现时，从屏幕右上角滑入（x:100%→0，opacity:0→1）弹簧动画；停留后滑出消失，点击关闭时触发缩小渐隐动画

    - 核心API：`spring`、`animate`

    - 价值：系统通知不突兀，不干扰用户当前操作，同时保证信息触达

3. **下拉菜单展开收起动画**

    - 适用场景：顶部管理员下拉菜单、子模块筛选下拉框

    - 效果说明：展开时，菜单高度从0平滑过渡到目标高度，内部菜单项从上到下依次渐入；收起时反向执行，高度收缩+菜单项渐隐

    - 核心API：`spring`、`animate`、`stagger`

    - 价值：下拉操作更丝滑，展开收起有明确层级感，提升界面操作质感

### 2. 性能优化强制要求

- 长页面滚动动画、循环微动效，在元素离开视口/页面失焦时自动暂停，降低CPU占用；

- 复杂动画用`will-change`提前优化，避免页面卡顿；

- 低性能设备自动降级为基础缓动动画，关闭弹簧动效与循环微动效，保障功能正常使用；

- 所有动画时长控制在200-600ms，避免过长动画影响操作效率。
