# 抻面综合评分体系

## 一、体系概述

本评分体系结合**关键帧标注评分**和**骨架线数据**，构建权威、可靠的综合评分系统。

### 1.1 数据来源

1. **关键帧标注评分**（36个关键帧）
   - 来源：cm1, cm2, cm3 标准视频的关键帧
   - 维度：
     - `noodle_rope`: thickness, elasticity, gloss, integrity
     - `hand`: position, action, angle, coordination
     - `noodle_bundle`: tightness, uniformity

2. **骨架线数据**（逐帧手部关键点）
   - 来源：MediaPipe HandLandmarker（21个关键点）
   - 数据：每帧的手部关键点坐标和置信度

### 1.2 评分原则

- **客观性**：基于可量化的特征和统计规律
- **权威性**：基于标准数据集（36个关键帧）建立基准
- **可靠性**：多重验证机制，降低误判率
- **可追溯性**：每个评分都有明确的公式和依据

## 二、骨架线特征提取

### 2.1 手部姿态特征

#### 2.1.1 手腕位置（Wrist Position）
```python
wrist_x = keypoints[0]['x']  # 手腕X坐标
wrist_y = keypoints[0]['y']  # 手腕Y坐标
wrist_confidence = keypoints[0]['confidence']
```

**用途**：评估手部位置（position）

#### 2.1.2 手掌方向（Palm Direction）
```python
# 基于小指根部和食指根部计算手掌方向
pinky_base = keypoints[17]  # 小指根部
index_base = keypoints[5]   # 食指根部

palm_direction_x = index_base['x'] - pinky_base['x']
palm_direction_y = index_base['y'] - pinky_base['y']
palm_angle = atan2(palm_direction_y, palm_direction_x) * 180 / π
```

**用途**：评估手部角度（angle）

#### 2.1.3 手指张开度（Finger Spread）
```python
# 计算各手指尖到手腕的距离
finger_distances = []
for finger_tip_idx in [4, 8, 12, 16, 20]:  # 拇指、食指、中指、无名指、小指
    tip = keypoints[finger_tip_idx]
    wrist = keypoints[0]
    distance = sqrt((tip['x'] - wrist['x'])**2 + (tip['y'] - wrist['y'])**2)
    finger_distances.append(distance)

finger_spread = std(finger_distances)  # 标准差，反映手指张开均匀度
```

**用途**：评估动作规范性（action）

#### 2.1.4 双手协调性（Bilateral Coordination）
```python
# 对于双手检测
if len(hands) == 2:
    left_wrist = hands[0]['keypoints'][0]
    right_wrist = hands[1]['keypoints'][0]
    
    # 计算双手距离
    hand_distance = sqrt((left_wrist['x'] - right_wrist['x'])**2 + 
                         (left_wrist['y'] - right_wrist['y'])**2)
    
    # 计算双手对称性（Y坐标差）
    symmetry_y_diff = abs(left_wrist['y'] - right_wrist['y'])
    
    # 计算双手角度一致性
    left_palm_angle = calculate_palm_angle(hands[0])
    right_palm_angle = calculate_palm_angle(hands[1])
    angle_consistency = 180 - abs(left_palm_angle - right_palm_angle)
```

**用途**：评估协调性（coordination）

### 2.2 时序特征（Temporal Features）

#### 2.2.1 手部速度（Hand Velocity）
```python
# 计算相邻帧之间的位移
if frame_idx > 0:
    prev_wrist = previous_frame['hands'][0]['keypoints'][0]
    curr_wrist = current_frame['hands'][0]['keypoints'][0]
    
    velocity_x = (curr_wrist['x'] - prev_wrist['x']) / frame_interval
    velocity_y = (curr_wrist['y'] - prev_wrist['y']) / frame_interval
    velocity_magnitude = sqrt(velocity_x**2 + velocity_y**2)
```

**用途**：评估动作流畅度（action）

#### 2.2.2 动作稳定性（Motion Stability）
```python
# 计算速度的标准差（窗口内）
velocity_window = [v1, v2, ..., vn]  # 最近N帧的速度
velocity_std = std(velocity_window)

# 稳定性评分：标准差越小，稳定性越高
stability_score = 1.0 / (1.0 + velocity_std)
```

**用途**：评估动作质量（action）

#### 2.2.3 动作连贯性（Motion Continuity）
```python
# 计算加速度（速度的变化率）
acceleration = (current_velocity - previous_velocity) / frame_interval

# 连贯性评分：加速度变化平滑
continuity_score = 1.0 / (1.0 + abs(acceleration))
```

**用途**：评估动作连贯性（action）

## 三、评分公式体系

### 3.1 手部评分公式

#### 3.1.1 位置评分（Position Score）

**公式1：基于骨架线的手腕位置**
```python
S_position_skeleton = 5.0 * (1.0 - min(1.0, position_deviation / max_deviation))
```

其中：
- `position_deviation` = 当前手腕位置与理想位置的距离
- `ideal_position` = (0.5 * width, 0.6 * height)  # 画面中心偏下
- `max_deviation` = sqrt(width² + height²) / 2  # 最大可能偏差

**公式2：基于关键帧标注的位置评分**
```python
S_position_annotation = annotation_scores['position']  # 直接使用标注评分
```

**综合公式：**
```python
S_position = α₁ × S_position_skeleton + α₂ × S_position_annotation

其中：
α₁ = 0.4  # 骨架线权重
α₂ = 0.6  # 标注权重（更可靠）
```

#### 3.1.2 角度评分（Angle Score）

**公式1：基于骨架线的手掌角度**
```python
# 计算手掌角度与标准角度的偏差
angle_deviation = abs(palm_angle - ideal_angle)
ideal_angle = 90°  # 标准抻面动作，手掌应接近水平

S_angle_skeleton = 5.0 * (1.0 - min(1.0, angle_deviation / 90.0))
```

**公式2：基于关键帧标注的角度评分**
```python
S_angle_annotation = annotation_scores['angle']
```

**综合公式：**
```python
S_angle = β₁ × S_angle_skeleton + β₂ × S_angle_annotation

其中：
β₁ = 0.5  # 骨架线权重（角度测量更准确）
β₂ = 0.5  # 标注权重
```

#### 3.1.3 动作评分（Action Score）

**公式1：基于骨架线的动作特征**
```python
# 综合多个特征
finger_spread_score = 5.0 * (1.0 - min(1.0, finger_spread / max_spread))
velocity_score = 5.0 * min(1.0, velocity_magnitude / ideal_velocity)
stability_score = 5.0 * motion_stability

S_action_skeleton = (finger_spread_score * 0.3 + 
                     velocity_score * 0.4 + 
                     stability_score * 0.3)
```

**公式2：基于关键帧标注的动作评分**
```python
S_action_annotation = annotation_scores['action']
```

**综合公式：**
```python
S_action = γ₁ × S_action_skeleton + γ₂ × S_action_annotation

其中：
γ₁ = 0.6  # 骨架线权重（动作特征更丰富）
γ₂ = 0.4  # 标注权重
```

#### 3.1.4 协调性评分（Coordination Score）

**公式1：基于骨架线的双手协调性**
```python
# 仅当检测到双手时计算
if len(hands) == 2:
    distance_score = 5.0 * (1.0 - min(1.0, abs(hand_distance - ideal_distance) / ideal_distance))
    symmetry_score = 5.0 * (1.0 - min(1.0, symmetry_y_diff / max_y_diff))
    angle_consistency_score = 5.0 * (angle_consistency / 180.0)
    
    S_coordination_skeleton = (distance_score * 0.3 + 
                               symmetry_score * 0.3 + 
                               angle_consistency_score * 0.4)
else:
    S_coordination_skeleton = 0.0  # 单手无法评估协调性
```

**公式2：基于关键帧标注的协调性评分**
```python
S_coordination_annotation = annotation_scores['coordination']
```

**综合公式：**
```python
S_coordination = δ₁ × S_coordination_skeleton + δ₂ × S_coordination_annotation

其中：
δ₁ = 0.5  # 骨架线权重
δ₂ = 0.5  # 标注权重
```

### 3.2 面条评分公式

#### 3.2.1 粗细评分（Thickness Score）

**公式1：基于骨架线的手部距离推断**
```python
# 双手距离可以间接反映面条拉伸程度
if len(hands) == 2:
    hand_distance = calculate_hand_distance(hands[0], hands[1])
    
    # 理想拉伸距离（基于标准数据集统计）
    ideal_stretch_distance = mean(standard_hand_distances)
    
    # 距离越大，面条越细（拉伸越充分）
    stretch_ratio = hand_distance / ideal_stretch_distance
    
    # 假设：拉伸比例与粗细成反比
    thickness_estimate = base_thickness / stretch_ratio
    
    S_thickness_skeleton = map_to_score(thickness_estimate, thickness_thresholds)
else:
    S_thickness_skeleton = 0.0
```

**公式2：基于关键帧标注的粗细评分**
```python
S_thickness_annotation = annotation_scores['thickness']
```

**综合公式：**
```python
S_thickness = ε₁ × S_thickness_skeleton + ε₂ × S_thickness_annotation

其中：
ε₁ = 0.3  # 骨架线权重（间接推断，可靠性较低）
ε₂ = 0.7  # 标注权重（直接观察，更可靠）
```

#### 3.2.2 弹性评分（Elasticity Score）

**公式1：基于骨架线的拉伸速度**
```python
# 计算双手距离的变化率（拉伸速度）
if len(hands) == 2 and frame_idx > 0:
    prev_distance = previous_hand_distance
    curr_distance = current_hand_distance
    
    stretch_velocity = (curr_distance - prev_distance) / frame_interval
    
    # 理想拉伸速度（基于标准数据集）
    ideal_stretch_velocity = mean(standard_stretch_velocities)
    
    # 速度接近理想值，弹性越好
    velocity_deviation = abs(stretch_velocity - ideal_stretch_velocity)
    S_elasticity_skeleton = 5.0 * (1.0 - min(1.0, velocity_deviation / ideal_stretch_velocity))
else:
    S_elasticity_skeleton = 0.0
```

**公式2：基于关键帧标注的弹性评分**
```python
S_elasticity_annotation = annotation_scores['elasticity']
```

**综合公式：**
```python
S_elasticity = ζ₁ × S_elasticity_skeleton + ζ₂ × S_elasticity_annotation

其中：
ζ₁ = 0.4  # 骨架线权重（时序特征）
ζ₂ = 0.6  # 标注权重
```

#### 3.2.3 光泽评分（Gloss Score）

**说明**：光泽主要基于图像特征，骨架线无法直接评估。

**公式：**
```python
S_gloss = S_gloss_annotation  # 完全依赖标注评分
```

#### 3.2.4 完整程度评分（Integrity Score）

**公式1：基于骨架线的动作连贯性推断**
```python
# 如果动作不连贯，可能导致面条断裂
if motion_continuity < threshold:
    S_integrity_skeleton = 3.0  # 降低评分
else:
    S_integrity_skeleton = 5.0  # 正常评分
```

**公式2：基于关键帧标注的完整程度评分**
```python
S_integrity_annotation = annotation_scores['integrity']
```

**综合公式：**
```python
S_integrity = η₁ × S_integrity_skeleton + η₂ × S_integrity_annotation

其中：
η₁ = 0.2  # 骨架线权重（间接推断）
η₂ = 0.8  # 标注权重（直接观察）
```

## 五、关键帧插值策略

### 5.1 关键帧识别

- **已标注关键帧**：36个关键帧（cm1: 9个, cm2: 8个, cm3: 19个）
- **关键帧索引**：存储在 `data/scores/抻面/{video}/key_frames.json`

### 5.2 插值方法

#### 5.2.1 时间插值（Temporal Interpolation）

对于非关键帧，使用前后关键帧的评分进行线性插值：

```python
def interpolate_score(frame_idx, key_frames, scores):
    # 找到前后关键帧
    prev_keyframe = find_previous_keyframe(frame_idx, key_frames)
    next_keyframe = find_next_keyframe(frame_idx, key_frames)
    
    if prev_keyframe and next_keyframe:
        # 线性插值
        alpha = (frame_idx - prev_keyframe['frame']) / \
                (next_keyframe['frame'] - prev_keyframe['frame'])
        
        interpolated_score = (prev_keyframe['score'] * (1 - alpha) + 
                              next_keyframe['score'] * alpha)
    elif prev_keyframe:
        # 只有前关键帧，使用前关键帧评分（降低置信度）
        interpolated_score = prev_keyframe['score'] * 0.9
    elif next_keyframe:
        # 只有后关键帧，使用后关键帧评分（降低置信度）
        interpolated_score = next_keyframe['score'] * 0.9
    else:
        # 无关键帧，使用骨架线评分
        interpolated_score = calculate_skeleton_score(frame_idx)
    
    return interpolated_score
```

#### 5.2.2 骨架线辅助评分

对于无关键帧标注的帧，完全依赖骨架线评分：

```python
def calculate_skeleton_score(frame_idx):
    # 提取当前帧的骨架线特征
    skeleton_features = extract_skeleton_features(frame_idx)
    
    # 使用标准数据集的统计信息进行评分
    skeleton_score = compare_with_standard(skeleton_features, standard_statistics)
    
    return skeleton_score
```

## 七、评分可靠性保证

### 7.1 多重验证机制

1. **置信度加权**：根据数据来源的可靠性分配权重
2. **异常检测**：识别异常值并进行修正
3. **一致性检查**：确保相邻帧评分的一致性

### 7.2 置信度计算

```python
def calculate_confidence(frame_idx, annotation_score, skeleton_score):
    # 基础置信度
    base_confidence = 1.0
    
    # 如果有标注评分，置信度提高
    if annotation_score is not None:
        base_confidence += 0.3
    
    # 如果骨架线数据完整，置信度提高
    if skeleton_data_complete(frame_idx):
        base_confidence += 0.2
    
    # 如果与标准数据集一致，置信度提高
    if matches_standard_pattern(frame_idx):
        base_confidence += 0.2
    
    return min(1.0, base_confidence)
```

### 7.3 异常值处理

```python
def detect_and_correct_outliers(scores, window_size=10):
    # 使用滑动窗口检测异常值
    for i in range(len(scores)):
        window = scores[max(0, i-window_size//2):min(len(scores), i+window_size//2)]
        mean_score = np.mean(window)
        std_score = np.std(window)
        
        if abs(scores[i] - mean_score) > 2 * std_score:
            # 异常值，使用中位数替换
            scores[i] = np.median(window)
    
    return scores
```
