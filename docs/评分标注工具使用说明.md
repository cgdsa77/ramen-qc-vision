# 抻面评分标注工具使用说明

## 一、工具概述

评分标注工具用于基于已标注的数据集（noodle_rope, hand, noodle_bundle）手动定义评分标准和判定细则。

## 二、访问方式

1. 启动Web服务：
   ```bash
   python start_web.py
   ```

2. 访问评分标注工具：
   ```
   http://localhost:8000/scoring-tool
   ```

## 三、使用流程

### 步骤1：选择视频
- 在"选择视频"下拉菜单中选择要评分的视频（如cm1, cm2, cm3等）
- 系统会自动加载该视频的所有帧

### 步骤2：浏览帧
- 使用"上一帧"/"下一帧"按钮或直接输入帧号浏览视频帧
- 点击"加载帧"按钮加载当前帧的检测结果

### 步骤3：查看检测框
- 图片上会显示已标注的检测框（不同颜色代表不同类别）
- 红色：hand（手部）
- 蓝色：noodle_rope（面条）
- 绿色：noodle_bundle（面条束）

### 步骤4：进行评分

#### 对于noodle_rope（面条），需要评分以下属性：
1. **粗细（Thickness）**：1-5分
   - 5分：粗细均匀，符合标准
   - 4分：基本均匀，略有偏差
   - 3分：存在明显不均匀
   - 2分：粗细差异较大
   - 1分：粗细严重不均

2. **弹性（Elasticity）**：1-5分
   - 5分：拉伸充分，弹性良好
   - 4分：拉伸基本到位
   - 3分：拉伸不够充分
   - 2分：拉伸不足
   - 1分：无明显拉伸

3. **光泽（Gloss）**：1-5分
   - 5分：表面光滑，有良好光泽
   - 4分：表面较光滑，有一定光泽
   - 3分：表面一般，光泽不明显
   - 2分：表面粗糙，缺乏光泽
   - 1分：表面粗糙，无光泽

4. **完整程度（Integrity）**：1-5分
   - 5分：面条完整，无断裂
   - 4分：基本完整，有轻微瑕疵
   - 3分：有少量断裂或破损
   - 2分：有明显断裂或破损
   - 1分：多处断裂，不完整

#### 对于hand（手部），需要评分以下属性：
1. **位置（Position）**：1-5分
   - 5分：位置完全正确，对称性好
   - 4分：位置基本正确，略有偏差
   - 3分：位置一般，对称性一般
   - 2分：位置偏差较大
   - 1分：位置错误

2. **动作（Action）**：1-5分
   - 5分：动作标准，符合规范
   - 4分：动作基本标准
   - 3分：动作一般
   - 2分：动作不规范
   - 1分：动作错误

3. **角度（Angle）**：1-5分
   - 5分：角度完全符合标准
   - 4分：角度基本符合
   - 3分：角度偏差在可接受范围
   - 2分：角度偏差较大
   - 1分：角度错误

4. **协调性（Coordination）**：1-5分
   - 5分：双手动作完全协调
   - 4分：双手动作基本协调
   - 3分：双手动作协调性一般
   - 2分：双手动作不够协调
   - 1分：双手动作不协调

#### 对于noodle_bundle（面条束），需要评分以下属性：
1. **紧密度（Tightness）**：1-5分
2. **均匀性（Uniformity）**：1-5分

### 步骤5：添加备注
- 在"备注"文本框中记录评分依据或特殊情况
- 例如："面条粗细约2.5mm，符合标准"、"手部位置略偏右"

### 步骤6：保存评分
- 完成当前帧的所有评分后，点击"💾 保存当前帧评分"按钮
- 评分数据会保存到 `data/scores/抻面/{视频名}/{帧名}_scores.json`

## 四、评分标准参考

详细的评分标准定义请参考：`docs/评分标准定义指南.md`

## 五、数据管理

### 查看评分进度
```bash
python scripts/manual_scoring_helper.py --action progress
```

### 分析评分数据
```bash
python scripts/manual_scoring_helper.py --action analyze
```

### 导出标准数据集
```bash
python scripts/manual_scoring_helper.py --action export
```

导出的数据集保存在：`data/scores/抻面/standard_dataset.json`

## 六、评分数据格式

每个帧的评分数据格式如下：
```json
{
  "video": "cm1",
  "frame": "cm1_00001.jpg",
  "scores": {
    "detection_0": {
      "thickness": 5,
      "elasticity": 4,
      "gloss": 5,
      "integrity": 5,
      "notes": "面条质量优秀"
    },
    "detection_1": {
      "position": 5,
      "action": 4,
      "angle": 5,
      "coordination": 4,
      "notes": "手部姿态良好"
    }
  },
  "timestamp": 1234567890.123
}
```

## 七、注意事项

1. **一致性**：确保评分标准一致，建议多人评分后取平均值
2. **完整性**：尽量覆盖所有关键动作阶段
3. **代表性**：优先评分标准视频（cm1, cm2, cm3）的关键帧
4. **可追溯性**：在备注中记录评分依据
5. **定期保存**：每完成一帧评分后及时保存

## 八、后续应用

完成评分后，可以使用导出的标准数据集：
1. 建立评分规则和阈值
2. 训练评分模型
3. 应用到新视频的自动评分
