# 评分规则使用说明

## 概述

基于标准数据集（cm1, cm2, cm3的36个关键帧）建立的评分规则和阈值，可用于对新视频进行自动评分。

## 评分规则文件

**位置**：`data/scores/抻面/scoring_rules.json`

**内容**：
- 各属性的评分阈值（优秀、良好、一般、较差、差）
- 各属性的统计信息（平均值、标准差、分布等）
- 权重配置（各类别和属性的权重）

## 评分阈值

### 面条（noodle_rope）属性阈值：

1. **粗细（thickness）**
   - 优秀(5分): >= 4.5
   - 良好(4分): 3.5 ~ 4.5

   - 差(1分): < 2.2

2. **弹性（elasticity）**
   - 优秀(5分): >= 4.5

   - 差(1分): < 2.5

3. **光泽（gloss）**
   - 优秀(5分): >= 4.5

   - 较差(2分): 2.5 ~ 2.9

4. **完整程度（integrity）**

   - 一般(3分): 3.1 ~ 3.8

## 权重配置

### 类别权重：
- 面条（noodle_rope）：60%
- 手部（hand）：30%
- 面条束（noodle_bundle）：10%

### 面条属性权重：
- 粗细：30%
- 弹性：25%
- 光泽：20%
- 完整程度：25%

### 手部属性权重：
- 位置：25%
- 动作：30%
- 角度：20%
- 协调性：25%

### 面条束属性权重：
- 紧密度：50%
- 均匀性：50%

## 使用方法

### 1. 通过Web API使用

访问：http://localhost:8000/web/video_detection.html

1. 上传视频
2. 点击"生成评分"按钮
3. 系统会自动使用评分规则进行评分

### 2. 通过Python代码使用

```python
from src.scoring.stretch_scorer import StretchScorer

# 初始化评分器
scorer = StretchScorer()

# 对检测结果进行评分
detections = [...]  # 检测结果列表
video_score = scorer.score_video(detections)

print(f"总体评分: {video_score['average_overall_score']}")
print(f"各类别评分: {video_score['class_average_scores']}")
```

## 评分结果说明

评分结果包含：
- `average_overall_score`: 总体平均分（0-5分）
- `class_average_scores`: 各类别平均分
- `frame_scores`: 各帧的详细评分
- `scored_frames`: 已评分的帧数

## 注意事项

1. **当前实现**：评分基于检测结果的置信度和基本特征，是简化版本
2. **未来改进**：可以基于图像特征（粗细、光泽等）进行更精确的评分
3. **阈值调整**：可以根据实际效果调整阈值和权重
4. **规则更新**：如果标准数据集更新，需要重新运行 `build_scoring_rules.py` 更新规则

## 更新评分规则

如果标准数据集有更新，可以重新建立评分规则：

```bash
python scripts/build_scoring_rules.py
```

这会重新分析标准数据集，更新评分阈值和统计信息。
