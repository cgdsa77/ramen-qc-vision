# 增强的抻面综合评分体系说明

## 一、概述

本评分体系基于论文《基于深度学习的乒乓球姿态动作评分方法》进行了优化，引入了以下核心方法：

1. **空间角度特征提取**：从骨骼关节点数据提取空间角度变化值
2. **改进的DTW算法**：用于匹配动作序列，计算序列距离
3. **Mean Shift算法**：确定关键关节点的权重系数
4. **基于序列距离的评分**：根据DTW距离计算评分值

---

## 二、核心方法

### 2.1 空间角度特征提取

#### 原理
从MediaPipe Hand的21个关键点提取空间角度特征，包括：
- **手指关节角度**：拇指、食指、中指、无名指、小指的各关节角度
- **手掌角度**：手掌方向角度、手掌基础角度
- **手指间角度**：手指张开角度
- **双手角度关系**：双手手腕连线角度、双手手掌角度差异

#### 优势
- **鲁棒性**：角度特征对位置变化不敏感
- **描述性**：能更好地描述动作姿态
- **可比较性**：角度值便于序列间比较

### 2.2 改进的DTW算法

#### 原理
动态时间规整（Dynamic Time Warping）用于匹配两个不同长度的动作序列：

1. **对齐路径**：找到两个序列之间的最佳对齐方式
2. **加权距离**：考虑关键关节点的权重
3. **归一化**：将距离归一化到可比较的范围

#### 改进点
- **窗口限制**：使用Sakoe-Chiba band限制搜索范围
- **加权距离**：根据关键关节点权重计算距离
- **多距离度量**：支持欧氏距离、曼哈顿距离、余弦距离

#### 公式
```
DTW距离 = min(Σ weighted_distance(seq1[i], seq2[j]))
```

其中：
- `weighted_distance` = Σ(特征距离 × 关键点权重)
- 对齐路径满足单调性和连续性约束

### 2.3 Mean Shift关键关节点权重

#### 原理
使用Mean Shift算法识别关键关节点：

1. **特征提取**：对每个关节点提取变化特征（幅度、频率、稳定性）
2. **聚类分析**：使用Mean Shift算法对关节点进行聚类
3. **权重计算**：根据聚类大小和变化幅度确定权重

#### 公式
```
权重(关节点i) = (聚类大小 / 总关节点数) × (1 + 平均变化幅度 / 180°)
```

#### 优势
- **自适应**：根据实际数据自动确定关键关节点
- **客观性**：基于数据统计，不依赖主观判断
- **可解释性**：权重有明确的统计意义

### 2.4 基于序列距离的评分

#### 原理
将DTW距离转换为0-5分的评分：

```
评分 = 5.0 × exp(-DTW距离 / 最大预期距离)
```

#### 特点
- **非线性映射**：使用指数函数，距离越小评分越高
- **可调参数**：最大预期距离可根据标准数据集调整
- **平滑过渡**：评分变化平滑，避免突变

## 三、实施步骤

### 3.1 第一步：提取标准角度序列和权重

```bash
python scripts/extract_standard_angle_sequences.py
```

**输出**：
- `data/scores/抻面/standard_angle_sequences.json`：标准角度序列摘要
- `data/scores/抻面/keypoint_weights.json`：关键关节点权重
- `data/scores/抻面/angle_change_statistics.json`：角度变化统计信息

### 3.2 第二步：运行增强评分

```bash
# 单个视频
python scripts/run_enhanced_scoring.py cm1

# 所有视频
python scripts/run_enhanced_scoring.py
```

**输出**：
- `reports/enhanced_scores/enhanced_score_{video_name}.json`：单个视频评分结果
- `reports/enhanced_scores/summary.json`：汇总结果

## 五、方法优势

### 5.1 相比传统方法

| 特性 | 传统方法 | 增强方法 |
|------|---------|---------|
| **特征** | 坐标位置 | 空间角度 |
| **比较** | 单帧比较 | 序列匹配 |
| **权重** | 固定权重 | 自适应权重 |
| **鲁棒性** | 对位置敏感 | 对位置不敏感 |
| **准确性** | 中等 | 高 |

### 5.2 论文方法的应用

1. **空间角度特征**：提高特征鲁棒性
2. **DTW序列匹配**：考虑动作的时序性
3. **Mean Shift权重**：自动识别关键关节点
4. **距离评分**：客观、可重复的评分方法

---

## 六、参数调整

### 6.1 DTW参数

- **window_size**：窗口大小，None表示无限制（默认）
- **distance_metric**：距离度量（'euclidean', 'manhattan', 'cosine'）

### 6.2 Mean Shift参数

- **bandwidth**：带宽参数，控制聚类粒度（默认0.5）

### 6.3 评分参数

- **max_expected_distance**：最大预期距离，用于评分转换（默认50.0）

## 七、验证方法

### 7.1 标准视频验证

在标准视频（cm1, cm2, cm3）上验证：
- DTW距离应该较小
- 评分应该较高（接近5分）
- 与标准序列的匹配度应该高

### 7.2 一致性检查

- 相似动作应该有相似的DTW距离
- 评分应该与专家主观评分一致
- 关键关节点权重应该合理
