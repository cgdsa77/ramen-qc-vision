# 抻面动作评分标准定义指南

## 概述

本文档用于指导如何基于已标注的数据集（noodle_rope, hand, noodle_bundle）手动定义评分标准和判定细则。

## 一、面条（noodle_rope）属性定义

### 1. 粗细（Thickness）
**定义**：面条的直径或宽度

**评分标准（1-5分）**：
- **5分（优秀）**：粗细均匀，符合标准（约2-3mm）
- **4分（良好）**：基本均匀，略有偏差（±0.5mm）
- **3分（一般）**：存在明显不均匀（±1mm）
- **2分（较差）**：粗细差异较大（±1.5mm）
- **1分（差）**：粗细严重不均（±2mm以上）

**测量方法**：
- 通过检测框的宽度（width）估算
- 结合图像中参考物（如手部）进行相对比较
- 记录检测框宽度像素值，转换为实际尺寸

### 2. 弹性（Elasticity）
**定义**：面条的拉伸状态和回弹能力

**评分标准（1-5分）**：
- **5分（优秀）**：拉伸充分，面条呈现良好弹性状态
- **4分（良好）**：拉伸基本到位，弹性良好
- **3分（一般）**：拉伸不够充分，弹性一般
- **2分（较差）**：拉伸不足，弹性较差
- **1分（差）**：无明显拉伸，缺乏弹性

**判断依据**：
- 面条的弯曲程度
- 拉伸长度与原始长度比例
- 面条的形态（是否呈现拉伸状态）

### 3. 光泽（Gloss）
**定义**：面条表面的光泽度和质感

**评分标准（1-5分）**：
- **5分（优秀）**：表面光滑，有良好光泽
- **4分（良好）**：表面较光滑，有一定光泽
- **3分（一般）**：表面一般，光泽不明显
- **2分（较差）**：表面粗糙，缺乏光泽
- **1分（差）**：表面粗糙，无光泽

**判断依据**：
- 图像中面条的亮度分布
- 表面反射光线的程度
- 与背景的对比度

### 4. 完整程度（Integrity）
**定义**：面条的连续性和完整性

**评分标准（1-5分）**：
- **5分（优秀）**：面条完整，无断裂
- **4分（良好）**：基本完整，有轻微瑕疵
- **3分（一般）**：有少量断裂或破损
- **2分（较差）**：有明显断裂或破损
- **1分（差）**：多处断裂，不完整

**判断依据**：
- 检测框内面条的连续性
- 是否存在明显的断裂点
- 面条的形态是否完整

## 二、手部（hand）姿态定义

### 1. 手部位置（Position）
**定义**：手部相对于面条的位置

**标准姿态**：
- **正确位置**：双手位于面条两侧，对称分布
- **位置评分（1-5分）**：
  - 5分：位置完全正确，对称性好
  - 4分：位置基本正确，略有偏差
  - 3分：位置一般，对称性一般
  - 2分：位置偏差较大
  - 1分：位置错误

### 2. 手部动作（Action）
**定义**：手部执行的具体动作

**标准动作类型**：
- **拉伸动作**：双手向外拉伸面条
- **旋转动作**：双手旋转面条
- **调整动作**：微调面条位置
- **抓握动作**：抓握面条

**动作评分（1-5分）**：
- 5分：动作标准，符合规范
- 4分：动作基本标准
- 3分：动作一般
- 2分：动作不规范
- 1分：动作错误

### 3. 手部姿态角度（Angle）
**定义**：手部与面条的角度关系

**标准角度**：
- **拉伸时**：双手与面条呈90度左右
- **旋转时**：双手与面条呈45-90度

**角度评分（1-5分）**：
- 5分：角度完全符合标准
- 4分：角度基本符合
- 3分：角度偏差在可接受范围
- 2分：角度偏差较大
- 1分：角度错误

### 4. 双手协调性（Coordination）
**定义**：双手动作的协调程度

**评分标准（1-5分）**：
- **5分（优秀）**：双手动作完全协调，同步性好
- **4分（良好）**：双手动作基本协调
- **3分（一般）**：双手动作协调性一般
- **2分（较差）**：双手动作不够协调
- **1分（差）**：双手动作不协调

## 三、面条束（noodle_bundle）属性

### 1. 束的紧密度（Tightness）
**定义**：面条束的紧密程度

**评分标准（1-5分）**：
- 5分：紧密适中，符合标准
- 4分：基本紧密
- 3分：紧密程度一般
- 2分：过于松散或过于紧密
- 1分：不符合标准

### 2. 束的均匀性（Uniformity）
**定义**：面条束的均匀分布

**评分标准（1-5分）**：
- 5分：分布均匀
- 4分：基本均匀
- 3分：均匀性一般
- 2分：不均匀
- 1分：严重不均匀

## 四、评分数据收集流程

### 步骤1：选择标准视频
从已标注的视频中选择3-5个作为"标准视频"（如cm1, cm2, cm3）

### 步骤2：逐帧评价
1. 打开评分标注工具
2. 选择视频和帧
3. 查看检测框
4. 对每个检测框的属性进行评分
5. 记录手部姿态信息

### 步骤3：建立评分数据库
将所有评分结果保存为JSON格式，作为标准数据集

### 步骤4：统计分析
- 计算各属性的平均值、标准差
- 确定评分阈值
- 建立评分规则

### 步骤5：验证和应用
- 用标准数据集验证评分规则
- 应用到新视频的自动评分

## 五、评分权重建议

### 面条（noodle_rope）权重分配：
- 粗细：30%
- 弹性：25%
- 光泽：20%
- 完整程度：25%

### 手部（hand）权重分配：
- 位置：25%
- 动作：30%
- 角度：20%
- 协调性：25%

### 总体评分权重：
- 面条质量：60%
- 手部姿态：40%

## 六、注意事项

1. **一致性**：确保评分标准一致，建议多人评分后取平均值
2. **代表性**：选择具有代表性的视频和帧进行评分
3. **完整性**：确保覆盖所有关键动作阶段
4. **可追溯性**：记录评分依据和备注信息
5. **可更新性**：评分标准可以根据实际情况进行调整
