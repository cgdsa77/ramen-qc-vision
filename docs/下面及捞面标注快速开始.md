## 📝 开始标注

### 方法1：使用X-AnyLabeling（推荐）

1. **打开X-AnyLabeling**

2. **设置标注格式**
   - 选择格式：YOLO
   - 选择类别文件：`data/labels/下面及捞面/classes.txt`

3. **打开图片目录**
   - 选择一个视频的图片目录，例如：`data/processed/下面及捞面/xl1/`

4. **设置保存路径**
   - 标注文件保存路径：`data/labels/下面及捞面/xl1/`
   - 确保每个视频的标注文件保存在对应的子目录中

5. **开始标注**
   - 按快捷键`W`创建边界框
   - 选择对应的类别
   - 保存标注（标注文件会自动保存）

### 方法2：使用labelImg（需要谨慎）

**⚠️ 重要提醒**：如果使用labelImg，请务必：

1. **先备份**：标注前运行备份脚本
   ```bash
   python scripts/backup_labels.py
   ```

2. **检查classes.txt**：每个视频子目录的`classes.txt`必须与主文件完全一致

3. **避免类别映射问题**：标注时不要随意添加或删除类别，严格按照主`classes.txt`的类别顺序

## 📌 标注注意事项

### 类别说明

1. **noodle_rope（面条）**
   - 标注正在捞出、已捞出在漏勺中、或其他情况下的面条
   - **注意**：在汤锅中的面条应标注为`soup_noodle`，不要标注为`noodle_rope`

2. **hand（手）**
   - 标注操作者的手部
   - 包括整个手部区域（含手腕）

3. **tools_noodle（工具）**
   - 标注捞面工具（漏勺、筷子等）
   - 标注整个工具的可见部分

4. **soup_noodle（汤中面条）**
   - 标注在汤锅中的面条
   - 用于区分汤中的面条和其他情况下的面条

### 标注要求

- ✅ 边界框要紧密贴合目标物体
- ✅ 确保每个目标都被标注（不要遗漏）
- ✅ 标注顺序要正确（类别ID必须与classes.txt一致）
- ✅ 标注文件保存为YOLO格式（.txt文件）

## ✅ 标注完成后的步骤

### 1. 验证标注一致性

运行验证脚本，确保所有classes.txt文件一致：

```bash
python scripts/verify_classes_consistency.py
```

（注意：这个脚本需要适配"下面及捞面"目录，或者手动检查）

### 2. 备份标注数据

```bash
python scripts/backup_labels.py
```

### 3. 训练模型

标注完成后，使用训练脚本训练模型：

```bash
python src/training/train_detection_model.py \
    --videos xl1 xl2 xl3 ... \
    --epochs 200 \
    --batch 8 \
    --patience 30 \
    --no-resume
```

（注意：训练脚本需要适配"下面及捞面"阶段）
