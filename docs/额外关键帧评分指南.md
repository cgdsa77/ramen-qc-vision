## 评分步骤

### 方法1：使用评分工具（推荐）

1. **访问评分工具**
   - 打开浏览器访问：http://localhost:8000/scoring-tool
   - 或双击运行：`打开评分工具.bat`

2. **选择视频和帧**
   - 在"选择视频"下拉框中选择要评分的视频（如cm1）
   - **勾选"仅显示关键帧"**（必须勾选）
   - **勾选"显示额外关键帧（新增）"**（重要！）
   - 系统会自动加载该视频的关键帧+额外关键帧

3. **进行评分**
   - 使用"上一帧"/"下一帧"按钮浏览帧
   - 为每个帧的检测项进行评分：
     - **noodle_rope**: 粗细、弹性、光泽、完整性（1-5分）
     - **hand**: 位置、动作、角度、协调性（1-5分）
     - **noodle_bundle**: 紧实度、均匀度（1-5分）**（重点！）**
   - 填写备注（可选）
   - 点击"保存评分"按钮

4. **切换到下一个视频**
   - 重复步骤2-3，为所有7个视频的额外关键帧评分

### 方法2：手动输入帧号

如果评分工具无法正确加载额外关键帧，可以手动输入帧号：

1. 访问评分工具
2. 选择视频
3. **取消勾选"仅显示关键帧"**（显示所有帧）
4. 在"帧号"输入框中输入帧的索引号（从汇总文件中查找）
5. 点击"加载帧"按钮
6. 进行评分

## 评分重点

### 优先评分包含noodle_bundle的帧

这些帧对于提高`tightness`和`uniformity`属性的样本数非常重要：

- **cm1**: 全部8帧都需要评分（全部含noodle_bundle）
- **cm2**: 优先评分前5帧（含noodle_bundle）
- **cm3**: 全部8帧都需要评分（全部含noodle_bundle）
- **cm4**: 优先评分前3帧（含noodle_bundle）
- **cm5**: 优先评分前4帧（含noodle_bundle）
- **cm6**: 无noodle_bundle帧，可以最后评分
- **cm7**: 优先评分前4帧（含noodle_bundle）

### 评分标准

参考文档：`docs/评分标准定义指南.md`

**特别注意noodle_bundle的评分**：
- **紧实度（tightness）**: 评估面条束的紧实程度
  - 5分：非常紧实，面条紧密贴合
  - 4分：较紧实，基本贴合
  - 3分：一般，略有松散
  - 2分：较松散，明显分离
  - 1分：很松散，面条分散

- **均匀度（uniformity）**: 评估面条束的均匀程度
  - 5分：非常均匀，粗细一致
  - 4分：较均匀，基本一致
  - 3分：一般，略有差异
  - 2分：不均匀，明显差异
  - 1分：很不均匀，差异很大
