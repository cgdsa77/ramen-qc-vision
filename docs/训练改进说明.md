# 训练改进说明

## 新增功能

### 1. 支持更大的模型

现在可以使用更大的YOLO模型进行训练，通常能获得更高的准确率：

- **YOLOv8n** (nano): 最小最快，适合快速测试
- **YOLOv8s** (small): **推荐**，准确率和速度的平衡
- **YOLOv8m** (medium): 更高准确率，但训练更慢
- **YOLOv8l** (large): 更高准确率，但训练更慢
- **YOLOv8x** (xlarge): 最高准确率，但训练非常慢

### 2. 支持GPU训练

现在可以自动检测并使用GPU进行训练，大幅提升训练速度：

- **CPU训练**: 约5-6小时（150轮）
- **GPU训练**: 约1-2小时（150轮）

### 3. 使用方法

#### 基本使用（使用GPU和YOLOv8s，推荐）

```bash
python src/training/train_detection_model.py --videos cm1 cm2 cm3 cm4 cm5 cm6 cm7 --model-size s --device auto
```

#### 使用更大的模型（YOLOv8m）

```bash
python src/training/train_detection_model.py --videos cm1 cm2 cm3 cm4 cm5 cm6 cm7 --model-size m --device auto
```

#### 强制使用CPU（如果GPU有问题）

```bash
python src/training/train_detection_model.py --videos cm1 cm2 cm3 cm4 cm5 cm6 cm7 --model-size s --device cpu
```

#### 所有参数

```bash
python src/training/train_detection_model.py \
    --videos cm1 cm2 cm3 cm4 cm5 cm6 cm7 \
    --epochs 200 \
    --batch 8 \
    --model-size s \
    --device auto \
    --lr0 0.01 \
    --patience 50
```

### 4. 参数说明

- `--model-size`: 模型大小 (n/s/m/l/x)，默认 s
- `--device`: 训练设备 (cpu/cuda/0/auto)，默认 auto（自动检测）
- `--epochs`: 训练轮数，默认 150
- `--batch`: 批次大小，默认 8
- `--lr0`: 初始学习率，默认 0.01
- `--patience`: 早停耐心值，默认 50

## 性能对比

| 模型 | 参数量 | 训练速度 | mAP50 (预估) |
|------|--------|----------|--------------|
| YOLOv8n | 3.2M | 最快 | 0.40-0.45 |
| YOLOv8s | 11.2M | 快 | **0.50-0.55** (推荐) |
| YOLOv8m | 25.9M | 中等 | 0.55-0.60 |
| YOLOv8l | 43.7M | 慢 | 0.60-0.65 |
| YOLOv8x | 68.2M | 很慢 | 0.65-0.70 |

## 建议

1. **首次训练**: 使用 `--model-size s --device auto`（YOLOv8s + GPU）
2. **如果准确率不够**: 尝试 `--model-size m`（YOLOv8m）
3. **如果GPU内存不足**: 减小 `--batch` 参数（如 `--batch 4`）
4. **如果训练太慢**: 使用 `--model-size n`（YOLOv8n，但准确率较低）
