# 评分体系优化总结

## 一、优化概述

基于论文《基于深度学习的乒乓球姿态动作评分方法》，对抻面评分体系进行了全面优化。

---

## 二、新增核心模块

### 2.1 空间角度特征提取模块 (`src/scoring/spatial_angle_extractor.py`)

**功能**：
- 从MediaPipe Hand 21个关键点提取空间角度特征
- 提取15个手指关节角度、2个手掌角度、3个手指间角度、2个双手角度关系
- 计算角度变化值（相邻帧之间的角度差）

**关键方法**：
- `extract_angles_from_frame()`: 从单帧提取角度
- `extract_angle_sequence()`: 从动作序列提取角度序列
- `calculate_angle_changes()`: 计算角度变化值

### 2.2 改进的DTW算法 (`src/scoring/improved_dtw.py`)

**功能**：
- 实现动态时间规整算法，用于匹配动作序列
- 支持加权距离计算（考虑关键关节点权重）
- 支持多种距离度量（欧氏距离、曼哈顿距离、余弦距离）
- 支持窗口限制（Sakoe-Chiba band）

**关键方法**：
- `compute()`: 计算两个序列的DTW距离
- `compute_angle_sequence_distance()`: 计算角度序列的DTW距离
- `normalize_distance()`: 归一化DTW距离

### 2.3 关键关节点权重估计器 (`src/scoring/keypoint_weight_estimator.py`)

**功能**：
- 使用Mean Shift算法识别关键关节点
- 基于聚类大小和变化幅度确定权重
- 自动从标准视频学习权重

**关键方法**：
- `mean_shift_clustering()`: Mean Shift聚类
- `calculate_keypoint_importance()`: 计算关键点重要性
- `estimate_weights_from_standard_videos()`: 从标准视频估计权重

### 2.4 增强的综合评分器 (`src/scoring/enhanced_comprehensive_scorer.py`)

**功能**：
- 集成空间角度特征、DTW算法、Mean Shift权重
- 基于DTW距离计算动作评分
- 融合标注评分和骨架线评分

**关键方法**：
- `calculate_dtw_score()`: 基于DTW计算评分
- `calculate_comprehensive_score()`: 计算综合评分
- `score_video()`: 对整个视频评分

## 四、核心公式

### 4.1 DTW距离计算

```
DTW距离 = min(Σ weighted_distance(seq1[i], seq2[j]))
```

其中：
- `weighted_distance` = Σ(特征距离 × 关键点权重)
- 对齐路径满足单调性和连续性约束

### 4.2 评分转换

```
评分 = 5.0 × exp(-DTW距离 / 最大预期距离)
```

### 4.3 关键关节点权重

```
权重(关节点i) = (聚类大小 / 总关节点数) × (1 + 平均变化幅度 / 180°)
```

## 五、优化效果

### 5.1 特征鲁棒性提升

- **原始方法**：基于坐标位置，对位置变化敏感
- **优化方法**：基于空间角度，对位置变化不敏感

### 5.2 评分准确性提升

- **原始方法**：单帧比较，忽略时序性
- **优化方法**：序列匹配（DTW），考虑动作的时序性

### 5.3 权重客观性提升

- **原始方法**：固定权重，依赖主观判断
- **优化方法**：自适应权重（Mean Shift），基于数据统计

## 六、使用流程

### 6.1 首次使用

1. **提取标准角度序列和权重**：
   ```bash
   python scripts/extract_standard_angle_sequences.py
   ```

2. **运行增强评分**：
   ```bash
   python scripts/run_enhanced_scoring.py cm1
   ```

### 6.2 日常使用

直接运行增强评分即可（权重已保存）：
```bash
python scripts/run_enhanced_scoring.py
```

## 八、技术优势

1. **权威性**：基于学术论文的方法，经过验证
2. **客观性**：基于数据统计，减少主观判断
3. **鲁棒性**：角度特征对位置变化不敏感
4. **准确性**：序列匹配考虑动作的时序性
5. **可解释性**：每个评分都有明确的公式和依据
