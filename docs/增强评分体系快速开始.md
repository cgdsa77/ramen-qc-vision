# 增强评分体系快速开始指南

## 一、概述

增强评分体系基于论文《基于深度学习的乒乓球姿态动作评分方法》，引入了：
- **空间角度特征**：更鲁棒的特征表示
- **DTW序列匹配**：考虑动作的时序性
- **Mean Shift权重**：自动识别关键关节点
- **距离评分**：客观、可重复的评分方法

---

## 二、快速开始（3步）

### 步骤1：提取标准角度序列和权重

```bash
python scripts/extract_standard_angle_sequences.py
```

**作用**：
- 从标准视频（cm1, cm2, cm3）提取角度序列
- 使用Mean Shift算法计算关键关节点权重
- 生成角度变化统计信息

**输出文件**：
- `data/scores/抻面/keypoint_weights.json`：关键关节点权重
- `data/scores/抻面/angle_change_statistics.json`：角度变化统计

### 步骤2：运行增强评分

```bash
# 单个视频
python scripts/run_enhanced_scoring.py cm1

# 所有视频
python scripts/run_enhanced_scoring.py
```

**输出文件**：
- `reports/enhanced_scores/enhanced_score_{video_name}.json`：评分结果

### 步骤3：查看结果

打开 `reports/enhanced_scores/summary.json` 查看汇总结果。

## 三、核心方法说明

### 3.1 空间角度特征

**提取的角度类型**：
- 手指关节角度（15个）
- 手掌角度（2个）
- 手指间角度（3个）
- 双手角度关系（2个）

**优势**：角度特征对位置变化不敏感，更适合动作比较。

### 3.2 DTW序列匹配

**原理**：找到两个动作序列之间的最佳对齐方式，计算加权距离。

**公式**：
```
DTW距离 = min(Σ weighted_distance(seq1[i], seq2[j]))
```

**评分转换**：
```
评分 = 5.0 × exp(-DTW距离 / 最大预期距离)
```

### 3.3 Mean Shift权重

**原理**：基于关节点变化特征的聚类，自动识别关键关节点。

**权重公式**：
```
权重 = (聚类大小 / 总关节点数) × (1 + 平均变化幅度 / 180°)
```

## 四、评分结果解读

### 4.1 DTW结果

- **distance**：DTW距离，越小越好
- **score**：基于距离的评分（0-5分）
- **matched_sequence**：匹配的标准序列索引

### 4.2 手部评分

- **position**：位置评分（基于角度特征）
- **angle**：角度评分（基于手掌角度）
- **action**：动作评分（基于DTW距离）
- **coordination**：协调性评分（基于双手角度一致性）

### 4.3 总分

综合所有维度的评分，范围0-5分。

## 六、常见问题

### Q1: 为什么需要先提取标准角度序列？

**A**: 标准角度序列用于：
1. 计算关键关节点权重
2. 作为DTW匹配的参考序列
3. 建立评分基准

### Q2: DTW距离如何转换为评分？

**A**: 使用指数衰减函数：
```
评分 = 5.0 × exp(-距离 / 50.0)
```

距离越小，评分越高。参数50.0可根据实际数据调整。

### Q3: 如何调整关键关节点权重？

**A**: 权重由Mean Shift算法自动计算，如需调整：
1. 修改 `scripts/extract_standard_angle_sequences.py` 中的 `bandwidth` 参数
2. 重新运行提取脚本
