评分标准说明

一、评分体系概述

本系统对抻面与下面及捞面两个环节分别维护一套评分规则。每套规则包含：各维度权重（决定各类别在总分中的占比）、评分行为参数（决定哪些检测参与计分、如何判定有效类别）、以及等级线（将百分制总分映射为优秀/良好/不及格）。

视频上传后，系统按帧检测目标（面绳、手部、面束等），对每帧按规则计算加权分，再在视频层面按有效类别加权得到平均分（1～5 分制），并换算为百分制与等级。以下说明各参数含义及对结果的影响。

二、各环节权重

权重表示该类别在视频级总分中所占比例，所有类别权重之和须为 100%。调整某一类权重会直接改变该类对最终分数的影响程度。

抻面：面绳、手部、面束三类；下面及捞面：面绳、手部、工具面、汤面四类。当前配置的可视化如下（打开说明时从当前规则读取）：

三、评分行为参数

  • 最低参与置信度 (0～1)：检测框置信度低于该值的框不参与该帧的评分。提高该值会过滤更多低置信度检测，分数更保守；降低则更多框参与，分数更容易受噪声影响。
  • 视频有效帧比例 (0～1)：某类别至少在视频总帧数中该比例以上的帧里出现，才会被计入视频级加权。例如 0.1 表示至少 10% 的帧出现该类才参与总分。提高该值会减少“偶现”类别对总分的影响。
  • 及格线 (0～100)：百分制总分 ≥ 及格线且 < 优秀线时，等级为「良好」；低于及格线为「不及格」。
  • 优秀线 (0～100)：百分制总分 ≥ 优秀线时，等级为「优秀」。优秀线须 ≥ 及格线。

四、百分制与等级换算

系统先将 1～5 分制的平均分线性换算为 0～100：百分制 = (平均分 - 1) / 4 × 100。再根据当前规则中的及格线、优秀线判定等级。

六、流程小结

① 每帧：按检测框置信度过滤（≥ 最低参与置信度）→ 按类别与属性权重得到该帧各类别分与总分。
② 视频：按「视频有效帧比例」确定有效类别 → 对有效类别按「各环节权重」加权平均得到 1～5 分制平均分。
③ 输出：平均分换算为百分制，再根据及格线/优秀线得到等级。保存规则后，下次评分将自动采用新参数。
